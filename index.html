Firebase connect केल्यावर — Mobile, Laptop, Tablet — सर्व devices वर data automatically sync होतो. Netlify वर deploy करा आणि mobile browser मध्ये उघडा.

⚫ Not Connected
🔗 Firebase Database URL
Firebase कसे setup करायचे (फ्री, 3 मिनिट):
1. console.firebase.google.com → Create project
2. Build → Realtime Database → Create Database
3. "Start in test mode" निवडा → Enable
4. Database URL copy करा (वरती paste करा) → Connect & Sync दाबा
⚠️ "Failed to fetch" error येत असेल तर:
Firebase Console → Realtime Database → Rules tab → हे paste करा:
{ "rules": { ".read": true, ".write": true } }
→ Publish दाबा → मग Connect & Sync दाबा

🚀 Netlify वर Deploy करा

एकदा Netlify वर deploy केल्यावर — तुमच्या mobile, laptop, client — सगळे एकाच link वर app उघडतात. Firebase connect असल्यास सर्वांना same data दिसतो.

Steps:
1. वरचे "Export Backup" दाबून data save करा
2. हे HTML file app.netlify.com/drop वर drag & drop करा
3. Netlify एक link देईल — तीच link सर्वांना share करा
4. Settings मध्ये Firebase URL connect करा → सर्व devices sync 🔄

☁️ Firebase Configuration
🔧 Setup (2 minutes, completely free):
  1. Go to console.firebase.google.com → Create project
  2. Build → Realtime Database → Create Database
  3. Choose "Start in test mode" → Enable
  4. Copy the Database URL shown (looks like:
    https://your-app-default-rtdb.firebaseio.com)
  5. Paste it below and click Connect
⚠️ Paste ONLY the URL (starting with https://) — no label text, no quotes
